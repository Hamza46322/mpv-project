# Usage

This guide covers how to use mpv to play media files, stream content, and navigate playback. Whether you're new to mpv or an advanced user, these instructions will help you leverage its capabilities as of May 2025.

# Prerequisites

Before using mpv, ensure:

  • mpv is installed (see Installation).
  • A terminal or command prompt is available (e.g., PowerShell on Windows, Terminal on macOS/Linux).
  • For streaming, install yt-dlp (pip install yt-dlp or via package manager, e.g., sudo apt install yt-dlp on Debian/Ubuntu).

# Basic Usage

mpv is a command-line media player with versatile playback options. Start with these essentials.

# Playing Local Files

Play a video or audio file:

mpv video.mp4

On Windows, use PowerShell or Command Prompt:

mpv "C:\Videos\movie.mp4"

# Streaming Online Content

Stream from URLs (e.g., YouTube) with yt-dlp installed:

mpv https://www.youtube.com/watch?v=example

Note Install yt-dlp for streaming support. See Tutorials for setup details.

# Playing Multiple Files

Load multiple files in a playlist:

mpv file1.mp4 file2.mkv

# Quitting mpv

Exit playback with:

  • q or Q (keyboard shortcut).
  • Close the terminal window (if no GUI is active).

# Common Options

Customize playback with these command-line options, which can be combined as needed.

Option Description Example Command
--vo=gpu or --vo=gpu-next Force GPU rendering (default or next-gen) mpv --vo=gpu video.mp4
--hwdec=auto Enable hardware decoding mpv --hwdec=auto video.mp4
--volume=150 Set volume to 150% (max 200%) mpv --volume=150 video.mp4
--speed=1.5 Play at 1.5x speed mpv --speed=1.5 video.mp4
--sub-file=subtitles.srt Load external subtitle file mpv --sub-file=subtitles.srt video.mp4
--sub-scale=1.2 Increase subtitle size by 20% mpv --sub-scale=1.2 video.mp4
--fullscreen Start in fullscreen mode mpv --fullscreen video.mp4
--ontop Keep window on top mpv --ontop video.mp4

# Example Command

Play a video with hardware decoding, fullscreen, and subtitles:

mpv --hwdec=auto --fullscreen --sub-file=subtitles.srt video.mp4

Note For a full list of options, run mpv --list-options or visit the mpv manual.

# Keyboard Shortcuts

mpv offers intuitive keyboard controls for efficient navigation. Default bindings are listed below, customizable via input.conf (see Configuration).

Action Key Description
Play/Pause SPACE Toggle playback
Seek Forward RIGHT Jump 10 seconds forward
Seek Backward LEFT Jump 10 seconds backward
Volume Up UP Increase volume by 5%
Volume Down DOWN Decrease volume by 5%
Fullscreen f Toggle fullscreen mode
Quit q or Q Exit mpv
Show Stats i Display real-time stats
Next/Prev File PGUP/PGDN Switch to next/previous file

Tip Press i during playback to view performance stats, useful for diagnosing issues.

# On-Screen Controller (OSC)

The On-Screen Controller (OSC) provides a GUI-like interface for mouse-based control, ideal for users who prefer visual navigation.

# Enabling OSC

The OSC is enabled by default. To ensure it’s active, add to mpv.conf:

osc=yes

# Using the OSC

  • Hover Over Video: The OSC appears at the bottom, showing play/pause, seek bar, volume, and time.
  • Click Controls: Click icons to play/pause, seek, or adjust volume.
  • Right-Click: Access a context menu for subtitles, audio tracks, or settings.

# Customizing OSC

  • Adjust visibility: osc-visibility=auto (shows on hover) or osc-visibility=always.
  • Install skins like uosc or ModernX for a modern look.

# Troubleshooting Usage Issues

Issue Solution
No Playback Verify file/URL path. Test with mpv --no-video for audio-only playback.
Black Screen Disable hardware decoding: mpv --hwdec=no. Check GPU compatibility.
Stuttering Reduce quality: mpv --scale=bilinear. Lower cache: --cache-secs=10.
OSC Not Showing Ensure osc=yes in mpv.conf or run mpv --osc. Verify GUI environment.
Streaming Fails Confirm yt-dlp is installed and updated: pip install --upgrade yt-dlp.
