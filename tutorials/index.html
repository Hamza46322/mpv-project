# Streaming YouTube

Stream YouTube videos directly in mpv using yt-dlp for high-quality playback.

# Setup

  1. Install yt-dlp:
    • Windows: In PowerShell, run pip install yt-dlp.
    • macOS/Linux: Use sudo apt install yt-dlp (Debian/Ubuntu) or brew install yt-dlp (macOS).
  2. Verify installation: Run yt-dlp --version to confirm.

# Streaming a Video

Play a YouTube video:

mpv https://www.youtube.com/watch?v=VIDEO_ID

Replace VIDEO_ID with the video’s ID (e.g., dQw4w9WgXcQ).

# Controlling Stream Quality

Select a specific resolution to optimize performance:

mpv --ytdl-format="bestvideo[height<=1080]+bestaudio/best" https://www.youtube.com/watch?v=VIDEO_ID

For persistent quality settings, add to mpv.conf:

ytdl-format=bestvideo[height<=1080]+bestaudio/best

Tip Update yt-dlp regularly with pip install --upgrade yt-dlp to ensure compatibility.

Warning Invalid URLs or outdated yt-dlp may cause streaming errors.

# Hardware Acceleration

Optimize performance by enabling hardware decoding, which offloads video processing to your GPU, as introduced in Configuration.

# Choosing the Right API

Select the appropriate hardware decoding API for your platform and GPU:

Platform Recommended API Command
Windows d3d11va (NVIDIA/AMD) --hwdec=d3d11va
Linux vaapi (Intel/AMD) --hwdec=vaapi
macOS videotoolbox --hwdec=videotoolbox

For automatic selection, add to mpv.conf:

hwdec=auto
vo=gpu

Or specify on the command line:

mpv --hwdec=auto --vo=gpu video.mp4

# Testing Hardware Acceleration

  1. Play a video with hardware acceleration enabled: 
    mpv --hwdec=auto video.mp4
  2. During playback, press the i key to cycle through on-screen display (OSD) information levels.
  3. Look for indicators of hardware decoding:
    • hwdec: yes or hwdec: no
    • The specific decoder in use (e.g., d3d11va, vaapi, videotoolbox)
  4. If hwdec: yes and the decoder name appear, hardware acceleration is active.

Note If hwdec: no is shown, try specifying the decoder explicitly (e.g., --hwdec=d3d11va on Windows).

Warning Hardware decoding may fail on older GPUs. Disable with --hwdec=no if issues occur.

# Managing Subtitles

Customize subtitles for clarity or adjust timing for synchronization, building on Usage.

# Loading Subtitles

  • Automatic Loading: Place subtitle files (e.g., video.srt) in the same directory as the video. mpv loads them if sub-auto=fuzzy is set in mpv.conf.
  • Manual Loading: Specify a subtitle file: 
    mpv --sub-file=subtitles.srt video.mp4

# Customizing Subtitles

Modify appearance in mpv.conf:

sub-font="Arial"        # Font style
sub-font-size=48        # Font size
sub-color=#FFFFFF       # Text color
sub-bold=yes            # Bold text
sub-margin-y=40         # Vertical margin
sub-border-size=2       # Border thickness

# Adjusting Subtitle Timing

Fix out-of-sync subtitles:

  • Delay subtitles by 0.5 seconds: 
    mpv --sub-delay=0.5 video.mp4
  • Advance subtitles by 0.5 seconds: 
    mpv --sub-delay=-0.5 video.mp4

During playback, use:

  • z: Increase subtitle delay by 0.1 seconds.
  • Z: Decrease subtitle delay by 0.1 seconds.

Tip Add sub-delay=0.5 to mpv.conf for persistent timing adjustments.

# Troubleshooting

Issue Solution
Streaming Fails Update yt-dlp: pip install --upgrade yt-dlp. Verify URL.
No Subtitles Loaded Ensure sub-auto=fuzzy in mpv.conf or use --sub-file.
Hardware Decoding Fails Try specific API: --hwdec=vaapi, etc. Check GPU support.
Playback Lags Lower streaming resolution: --ytdl-format=bestvideo[height<=720].
